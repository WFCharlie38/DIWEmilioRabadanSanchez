Práctica 1: Imágenes optimizadas para la web

Introducción

En esta práctica, trabajarás con imágenes optimizadas, un logotipo SVG y realizarás varias tareas relacionadas con licencias, formatos y optimización de imágenes para la web.

1. Investigación de licencias

Definición breve de licencia: Una licencia determina cómo puede utilizarse una obra (imagen, audio, vídeo o texto), especificando los permisos concedidos por el autor.

Diferencia entre Copyright y Creative Commons: El Copyright otorga al autor todos los derechos sobre su obra por defecto, mientras que las licencias Creative Commons permiten que el creador otorgue ciertos permisos para su uso, modificación o distribución.

Bancos de recursos libres

Banco Tipo de recursos Licencia
Unsplash Fotografías Licencia Unsplash (uso libre con restricciones mínimas)
Pexels Fotos y vídeos Licencia Pexels (uso gratuito y sin atribución obligatoria)
Pixabay Imágenes, vectores y vídeos Licencia Pixabay (similar a CC0)

2. Formatos de imagen para la web

Formato Peso medio Transparencia Escalabilidad Uso recomendado
JPG Bajo No No Fotografías y contenido con muchos colores
PNG Medio/Alto Sí No Gráficos con transparencia, iconos simples
SVG Muy bajo Sí Sí (vectorial) Logotipos, ilustraciones y gráficos escalables
WebP Muy bajo Sí No Imágenes optimizadas modernas (reemplazo de JPG/PNG)
